É uma API REST de clínica odontológica para cadastrar e gerenciar dentistas, procedimentos, materiais e consultas em um banco SQLite.

Ela expõe endpoints HTTP (GET, POST, PUT, DELETE) para permitir que um sistema externo ou o Postman crie, liste, atualize e exclua esses registros, salvando tudo no arquivo de banco odonto.db via Entity Framework Core.
